# PulseMetrics: Healthcare Operations & Marketing Analytics Pipeline

## 📌 Project Overview
PulseMetrics is a relational database ecosystem and analytical pipeline designed to showcase data-driven insights for hospital operations, pharmacy metrics, and patient engagement marketing, paired with custom dashboard visual wireframes.

---

## 🛠️ Tech Stack & Skills Showcased
* **Database Engine:** MySQL
* [cite_start]**SQL Skills:** Advanced Window Functions, Horizontal Joins, Table Constraints, Views, and Schema Architecture[cite: 1, 6, 11, 21, 27, 30].
* [cite_start]**Design/UX Tools:** Adobe Photoshop / Illustrator / Canva (used for data visualization mockups)[cite: 75, 83].

---

## 📁 Repository Structure
* `/schema/`: Contains scripts for database initialization, tables creation (DDL), and integrity constraints.
* `/queries/`: Advanced SQL analytics scripts covering pharmacy performance, patient retention, and marketing ROI.
* `/design_assets/`: UI/UX wireframes demonstrating how this data looks on an administrative dashboard.

# 🎨 PulseMetrics Admin Dashboard Design Wireframes

Welcome to the visual layer of the PulseMetrics platform! This folder bridges the gap between complex SQL database queries and clean, intuitive administrative tools.

## 📐 The Wireframe Layout Strategy
As an analytics professional with a design foundation, I designed a mock web application interface tailored for hospital administrators:
* **Top Metric Cards:** Driven by simple `COUNT` and `SUM` data values.
* **The Performance Chart:** A beautiful visual graph mapping out the `channel_rank` from our marketing SQL queries.
* **The Low-Stock Indicator Grid:** A high-visibility table interface powered by our `CASE WHEN stock_quantity = 0` database flags.
